# Feature Checks

This folder contains a deterministic end-to-end feature regression pack for the Web UI backend and frontend build.

- `inputs/`: request payloads and uploaded sample files
- `expected/`: expected responses for each exercised feature
- `actual/`: generated by the runner with real outputs from this workspace
- `run_feature_checks.py`: executes every feature once and validates outputs

Run it from the repository root with:

```powershell
.\.venv\Scripts\python.exe .\test\feature_checks\run_feature_checks.py
```

The runner uses an isolated SQLite database under `actual/` and patches IDs/timestamps so the outputs are stable and reviewable.
